# Gitignore Hub

![YAML](https://img.shields.io/badge/YAML-1.2-blue.svg) ![License](https://img.shields.io/badge/License-MIT-yellow.svg)

Curated collection of .gitignore templates.

## Features

- 20+ language and framework templates
- CLI generator tool
- Customizable output
- Always up-to-date patterns

## Quick Start

```bash
pip install -r requirements.txt
python generator.py --template python
python generator.py --list
python generator.py --template node docker -o .gitignore
```
